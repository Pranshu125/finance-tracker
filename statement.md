# Project Statement: Personal Finance Tracker

## Problem Statement

Students and young professionals struggle to track daily finances effectively. Existing tools are often too complex or require internet access. This leads to:

- Poor awareness of spending habits
- Difficulty in budget management  
- No systematic way to track income/expenses
- Financial records kept in scattered formats

## Scope of the Project

### What's Included:
- **Transaction Management**: Add/edit income and expenses with categories
- **Financial Reports**: View totals, balances, and category-wise breakdowns
- **Data Storage**: Save transactions in CSV files
- **CLI Interface**: Simple text-based menu system

### What's Not Included:
- Multi-user accounts
- Cloud sync or mobile app
- Bank integration
- Advanced charts or PDF reports

## Target Users

### Primary Users:
1. **College Students**
   - Need to track pocket money and expenses
   - Prefer simple, free tools
   - Basic computer skills

2. **Young Professionals**  
   - Want to monitor salary and spending
   - Value privacy and offline use
   - Limited time for complex tools

## High-Level Features

1. **Add Transactions**
   - Record income and expenses
   - Categorize transactions
   - Add descriptions and dates

2. **View Financial Summary**  
   - See total income and expenses
   - Check current balance
   - View category-wise spending

3. **Data Management**
   - Automatic saving to CSV file
   - Load previous transactions
   - Basic error handling

4. **Simple Interface**
   - Easy menu navigation
   - Clear instructions
   - Fast operation
