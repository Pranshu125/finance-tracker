# Personal Finance Tracker

## Overview
A simple CLI-based application to track personal income and expenses.

## Features
- Add income/expense transactions
- View financial summary
- Category-wise spending analysis
- Data storage in CSV format

## Technologies Used
- Python 3
- CSV file storage

## Installation & Run
1. Clone repository: `git clone [your-repo-url]`
2. Navigate to folder: `cd personal-finance-tracker`
3. Run: `python main.py`

## Testing
Run the program and use menu options to test all features.
