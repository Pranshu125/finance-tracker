# Personal Finance Tracker

## Overview
A simple CLI-based application to track personal income and expenses. Helps users manage their budget effectively.

## Features
- Add income/expense transactions
- View financial summary with balance
- Category-wise tracking
- Data persistence using CSV files
- Simple text-based interface

## Technologies Used
- Python 3
- CSV file storage
- Built-in Python libraries (csv, os)

## Installation & Run
1. Clone repository: `git clone https://github.com/Pranshu125/finance-tracker.git`
2. Navigate to folder: `cd finance-tracker`
3. Run: `python main.py`

## Testing
Run the program and use the menu to:
1. Add income transactions
2. Add expense transactions  
3. View summary to see balance
4. Check if data saves between sessions
